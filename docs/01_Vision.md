# Madinah Path

Version : Living Document
Status  : Active

---

# 1. Vision

Madinah Path adalah aplikasi pendamping keluarga muslim untuk membantu orang tua membimbing anak dalam membangun kebiasaan ibadah, hafalan Al-Qur'an, adab, dan pendidikan Islam secara bertahap sejak usia dini hingga dewasa.

Aplikasi ini bukan sekadar aplikasi checklist, tetapi menjadi jurnal perjalanan pendidikan Islam dalam keluarga.

---

# 2. Target User

Primary User

- Ummi
- Abi
- Wali Anak

Secondary User

- Anak

---

# 3. Core Values

- Mudah digunakan
- Fokus kepada keluarga
- Islami
- Tidak berlebihan
- Membangun kebiasaan
- Memotivasi
- Mudah dikembangkan

---

# 4. Long Term Goal

Membangun platform keluarga muslim yang mampu membantu orang tua dalam:

- Mengelola data keluarga
- Mengelola perkembangan setiap anak
- Hafalan
- Ibadah
- Target
- Reward
- Pendidikan
- Statistik
- Family Challenge
- Family Achievement

---

# 5. Design Philosophy

Semua fitur harus menjawab pertanyaan:

"Apakah fitur ini membantu orang tua membimbing anak menjadi muslim yang lebih baik?"

Jika tidak, fitur tersebut tidak dibuat.

---

# 6. Development Principle

Selalu membangun pondasi terlebih dahulu.

Urutan:

Foundation
↓

Feature
↓

Integration
↓

Testing
↓

Documentation
↓

Release

---

# 7. Out of Scope

Aplikasi ini bukan:

- Media sosial
- Game
- Marketplace
- Chatting
- Streaming

Fokus utama tetap sebagai aplikasi pendamping keluarga muslim.
